ProjetoPHP - Agendamento de consultorias de investimentos

O projeto consiste no desenvolvimento de um site voltado ao agendamento de consultorias de investimentos, permitindo que clientes interessados em orientação financeira possam marcar atendimentos de forma prática e automatizada. A plataforma busca oferecer uma experiência simples e intuitiva, tanto para o cliente quanto para o consultor, centralizando o processo de agendamento e gerenciamento de horários.

Descrição:
- Página inicial, usada para se direcionar as outras páginas do projeto.
- Página de agendamento, contendo o formulário para receber as informações e agendar a consultoria.
- Página com informações sobre os seus agendamentos, acessadas através do seu nome e CPF.
- Página de serviços, contendo uma pequena descrição sobre os principais planos da empresa.
- Página de fale conosco, usada para enviar reclamações, dúvidas, sugestões, etc.
- Painel de controle privado, usado para acesssar e gerenciar agendamentos, clientes e contatos.
